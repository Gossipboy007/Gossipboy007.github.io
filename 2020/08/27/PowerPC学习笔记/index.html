三目运算符

遇到以下汇编代码，直接使用三目运算符。参考： ARM/MIPS/PowerPC/RISC-V 指令集快速入门 | FANG - 🍉 (moooc.cc)

汇编指令对应C代码
subfic rA, rB, 0
adde rC, rA, rB		rC = (rB ? 0 : 1)
addic rA, rB, -1
subfe rC, rA, rB		rC = (rB ? 1 : 0)
addic rA, rB, -1
subfe rC, rA, rA		rC = (rB ? 0 : -1)
extsb rA, rA<br/>srawi rB, rA, 0x1F
xor rC, rB, rA
subf rC, rC, rB
srawi rC, rC, 0x1F

rC = (rA ? -1 : 0)
extsb rA, rA
xor rC, rA, rS
srawi rB, rC, 0x1F
xor rC, rB, rC
