这里解释一下这些是什么意思：

  1. ‘|’：，a | b的意思是将a的输出作为b的输入；
  2. xxd -p filename表示把filename进行十六进制存储；
  3. tr -d ‘\n’表示删除换行符；
  4. rev表示把字符相反输出；
  5. xxd -r -p表示把十六进制转换成二进制；
  6. 接着> reversed表示将其存储为名称为reversed的文件。
